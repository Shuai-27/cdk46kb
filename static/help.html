CDK4/6 Knowledge-Base 网站使用说明

1. 网站功能概览

  • Statistics 模块：展示 CDK4/6 知识库原始表格，可下载整库 CSV。
  • Global Network 模块：全局基因共现网络可视化，支持任意列模糊搜索，构建并渲染子网。
  • Centrality 模块：展示中心性指标（degree、betweenness、closeness 等）前 N 名基因的柱状图与 Venn 图。
  • Organic Framework 模块：可视化有机框架子网（直接从 organic_full.cyjs 中提取 elements）。
  • Subtype Networks 模块：展示乳腺癌不同亚型（Luminal B1、TNBC）对应的子网节点与边表，并可视化。

2. 示例数据 (Example Data)

点击下面链接可下载示例 CSV，用于快速体验各功能：

  • example_small.csv （示例数据含 5 行，字段：Gene Symbol 等）

3. API 接口示例

  • GET /api/stats → 返回整个 CDK4/6 知识库（JSON）
  • GET /api/centrality/degree?top=20 → 返回 degree 指标前 20 名基因（JSON）
  • GET /api/organic/elements → 返回从 organic_full.cyjs 中提取的 elements 列表（JSON）
  • GET /api/subtype/luminal_original/elements → 返回指定 subtype 的 elements

4. 数据下载
